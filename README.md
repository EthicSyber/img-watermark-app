# img-watermark-app [IN-PROGRESS]

This application is designed to place a watermark on any image you want to use for the purpose of watermarking it. This application uses the python PILLOW library for image process and will use Tkinter for application GUI development.

Currently the status is in-progress as I am still working on the watermarking aspect of the code.
